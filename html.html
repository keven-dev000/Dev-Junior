📄 Estrutura da Página

<!DOCTYPE html> = Diz ao navegador que o site é HTML5

<html> = Define o início do documento HTML

<head> = Contém metadados (título, links, scripts)

<title> = Define o título da aba do navegador

<body> = Contém todo o conteúdo visível da página

🧱 Estrutura de Conteúdo

<header> = Cabeçalho da página ou seção

<nav> = Menu de navegação

<main> = Conteúdo principal

<section> = Seção temática do conteúdo

<article> = Conteúdo independente (ex: post)

<aside> = Conteúdo lateral (ex: barra lateral)

<footer> = Rodapé da página ou seção

<div> = Container (sem significado semântico)

📋 Texto e Títulos

<h1> a <h6> = Títulos (do mais importante ao menos)

<p> = Parágrafo

<span> = trecho de texto sem significado semântico

<br> = Quebra de linha

<strong> = Texto importante (semântico + negrito)

<em> = Ênfase (semântico + itálico)

<b> = Negrito (sem semântica)

<i> = Itálico (sem semântica)

🔗 Links e Imagens

<a> = Link

<img> = Imagem

<figure> = Imagem com legenda

<figcaption> = Legenda da imagem

📰 Listas e Tabelas

<ul> = Lista não ordenada

<ol> = Lista ordenada

<li> = Item de lista

<table> = Tabela

<tr> = Linha da tabela

<td> = Célula da tabela

<th> = Cabeçalho da tabela

🗒 Formulários

<form> = Formulário

<input> = Campo de entrada

<label> = Rótulo de campo

<textarea> = Área de texto

<button> = Botão

<select> = Menu suspenso

<option> = Opção dentro do <select>
