💻 Estrutura do JavaScript

var, let, const = Declara variáveis (let e const são os mais usados)

function = Define uma função

return = Retorna um valor de uma função

if, else = Condicionais (executa algo se uma condição for verdadeira)

for, while = Estruturas de repetição (loops)

console.log() = Exibe informações no console (útil para testes)

📦 Tipos de Dados

String = "Olá mundo"

Number = 10, 3.14

Boolean = true, false

Array = [1, 2, 3]

Object = { nome: "Ana", idade: 20 }

null = Valor intencionalmente vazio

undefined = Variável sem valor atribuído

🧠 Funções e Eventos

addEventListener = Escuta eventos (ex: clique)

onclick, oninput, onchange = Executa código em eventos específicos

setTimeout() = Executa algo depois de um tempo

setInterval() = Executa algo repetidamente com intervalo

🔧 Manipulação de HTML (DOM)

document.getElementById() = Seleciona um elemento pelo ID

document.querySelector() = Seleciona com CSS (ex: .classe, #id)

element.innerHTML = Altera o conteúdo HTML

element.style.property = Altera o estilo via JavaScript

createElement() / appendChild() = Cria e adiciona elementos na página

🔄 Operadores e Comparações

+, -, *, /, % = Operações matemáticas

==, === = Compara valores (=== compara valor e tipo)

!=, !== = Verifica se são diferentes

>, <, >=, <= Comparações numéricas

🌐 Extras e Avançado

JSON.parse() / JSON.stringify() = Converte entre string e objeto

fetch() = Faz requisições HTTP (ex: APIs)

try { } catch { } = Trata erros

localStorage = Armazena dados no navegador

class = Cria objetos com estrutura (orientação a objetos)
