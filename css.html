🎨 Estrutura do CSS

color = Define a cor do texto

background-color = Define a cor de fundo

font-size = Tamanho do texto (ex: 16px, 1.2em)

font-family = Define a fonte usada no texto

text-align = Alinhamento do texto (left, center, right, justify)

font-weight = Espessura da fonte (normal, bold, lighter, 100-900)

line-height = Altura da linha (espaçamento entre linhas de texto)

📏 Espaçamento e Tamanho

margin = Espaço externo ao redor do elemento

padding = Espaço interno entre conteúdo e borda

width = Largura do elemento

height = Altura do elemento

max-width = Largura máxima permitida

min-height = Altura mínima permitida

box-sizing = Controla como a largura/altura é calculada (content-box, border-box)

🎯 Posição e Layout

display = Tipo de exibição (block, inline, flex, grid, none)

position = Tipo de posicionamento (static, relative, absolute, fixed, sticky)

top, right, bottom, left = Define posição com base no pai (quando position ≠ static)

z-index = Define a sobreposição (quem fica na frente)

overflow = Como lidar com conteúdo que "transborda" (hidden, scroll, auto)

✨ Efeitos Visuais

border = Adiciona borda ao redor do elemento

border-radius = Cantos arredondados

box-shadow = Sombra ao redor da caixa

box-shadow = Sombra ao redor da caixa

opacity = Transparência do elemento (0 a 1)

transition = Suaviza alterações visuais

transform = Move, gira, redimensiona elementos

filter = Aplica efeitos como blur ou brilho

📱 Responsividade

media queries = Define estilos para diferentes tamanhos de tela

flexbox (display: flex) = Layout flexível e responsivo

grid (display: grid) = Layout baseado em grade

vw, vh = Unidades relativas à tela (largura e altura da viewport)

% = Tamanho relativo ao pai
